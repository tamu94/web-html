Latitude vs X

The purpose of this project was to analyze how weather changes as you gary latitude. To accomplish the analysis, the OpenWeatherMAP API was used to assemble a dataset of 500 cities. Matplotlib was then utilized to demonstrate the various relationships of weather verus latitude. Factors examined included temperature, wind speed, cloudiness and humidity. This site provides the source data and visualizations created for the analysis. Trends and observations found are also discussed.
